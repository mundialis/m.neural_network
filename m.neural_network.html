NAME

m.neural_network - GRASS GIS addons to train and apply a neural network.

KEYWORDS

raster, vector

DESCRIPTION

The m.neural_network toolset consists of several modules.
  • m.neural_network.preparedata: Prepares and exports tiles for the label process
  • m.neural_network.preparedata.worker_export: Worker for parallel processing for exporting for m.neural_network.preparedata
  • m.neural_network.preparedata.worker_nullcells: Worker to analyse the number of null cells in parallel for m.neural_network.preparedata
  • m.neural_network.preparetraining: Prepares imagery and labelled data for training and application of a neural network.
  • m.neural_network.preparetraining: Worker to rasterize labelled data in parallel for m.neural_network.preparetraining<

REQUIREMENTS

The following Python libraries are needed.
  • grass-gis-helpers>=2.2.0
