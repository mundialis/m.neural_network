DESCRIPTION

m.neural_network.train trains or fine-tunes a neural network using the segmentation_models.pytorch framework for semantic segmentation.

A new model can be trained by specifying the options in_channels, out_classes model_arch, encoder_name, encoder_weights, in_channels. A locally saved model can be further trained (fine-tuned) by giving the path to the previously saved model. For more information about available encoder-decoder combinations, see the smp documentation.

A larger batchsize is generally better for training, but requires more GPU RAM. A bit of experimentation is needed to find a batch size that still fits into the GPU RAM.

NOTES

It is expected that all data lie in the directory structure and naming format as created by m.neural_network.preparedata.
