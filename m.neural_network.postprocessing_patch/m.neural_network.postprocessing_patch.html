DESCRIPTION

m.neural_network.postprocessing_patch patches the tiles (GeoTIFFs) which results from neural network inference.

EXAMPLE

Patch given list_of_files

m.neural_network.postprocessing_patch tiles_filelist=/path/to/list_of_files tiles_path=/path/to/classification/tifs edge_cut=12.8 output=classification_patch
