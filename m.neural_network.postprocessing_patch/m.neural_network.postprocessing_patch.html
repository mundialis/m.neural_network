DESCRIPTION

m.neural_network.postprocessing_patch patches the tiles (GeoTIFFs) which results from neural network inference.

EXAMPLE

Patch all files within given tiles_path with default edge_cut of 64 (Note: should be half of grid overlap)

m.neural_network.postprocessing_patch tiles_path=/path/to/classification/tifs output=classification_patch
